General commands

These commands can be used by any users in the same server of Sierra as long as they are enabled by the server owner

Name Usage Description
Help ;help This command can be used in a chat to show all available commands while in discord
Server info ;serverinfo This server will show all the info about the server you will need to know
Server invite ;serverinvite This command will send an invite to the current text channel which can be shared outside of the server
Bot info ;botinvite This command will show you all the info you will need to know about Sierra
Roles ;roles This command will display all roles on the server and which ones you have
User info ;userinfo or ;userinfo [MENTION] Will display all needed information about a user
Report ;report [ERROR OR ISSUE] This command logs the issue so that i can find and fix the problem

Admin commands

These commands can only be used by the admins of the server with the right server permissions

Name Usage Description
Announcement ;announce [MESSAGE] This command takes the message and posts it in the announcement channel
Clear ;clear [1 to 100] Deletes the amount of messages that is defined
Kick ;Kick [MENTION] [REASON] Will kick the mentioned user

Bot setup commands

These commands can only be used by admins with the administrator permission in the server

Name Usage Description
Toggle settings ;togglesettings Shows which commands and events are enabled or disabled
Channel settings ;channelsettings Shows what channels are being used for commands or events
Role settings ;rolesettings Shows what roles you'll need to use certain commands
Set color ;setcolor [COLOR] Changes the color of Sierra's embed messages
Rename ;rename [NAME] This command changes the nickname of Sierra in your server
Enable welcome messages ;enablewelcome This command when used in the channel you want welcome messages sent to will enable to event and setup the channel as the welcome channel
Enable leaving messages ;enablebye This command when used in the channel you want leaving messages sent to will enable to event and setup the channel as the leaving channel
Enable announce command ;enableannounce This command when used in the channel you want announce messages sent to will enable to event and setup the channel as the announce channel
Disable welcome messages ;disablewelcome This command when used will disable welcome messages for the server
Disable leaving messages ;disablebye This command when used will disable leaving messages for the server
Disable announce command ;disableannounce This command when used will disable the announcement command for the server

Sierra is always being updated and improved so more commands will be added as updates for Sierra are released.
