General commands

These commands can be used by any users in the same server of Sierra as long as they are enabled by the server owner

Name Usage Description
Help ;help [SECTION] This command will display all of the commands the bot has corresponding to the section you choose
Server info ;serverinfo This server will show all the info about the server you will need to know
Server invite ;serverinvite This command will send an invite to the current text channel which can be shared outside of the server
Bot info ;botinfo This command will show you all the info you will need to know about Sierra
Roles ;roles This command will display all roles on the server and which ones you have
User info ;userinfo or ;userinfo [MENTION] Will display all needed information about a user
Sierra invite ;addsierra This command sends you the link required to invite Sierra to your own server

Admin commands

These commands can only be used by the admins of the server with the right server permissions

Name Usage Description
Announcement ;announce [MESSAGE] This command takes the message and posts it in the announcement channel
Clear ;clear [1 to 100] Deletes the amount of messages that is defined
Kick ;Kick [MENTION] Will kick the mentioned user
ban ;Kick [MENTION] [REASON] Will ban the mentioned user
Unban ;Kick [USERS ID] Will unban the user with the specified ID

Bot setup commands

These commands are only available to people who have the permission manage guild

Name Usage Description
Toggle settings ;togglesettings Shows which commands and events are enabled or disabled
Channel settings ;channelsettings Shows what channels are being used for commands or events
Role settings ;rolesettings Shows what roles you'll need to use certain commands
Set color ;setcolor [COLOR] Changes the color of Sierra's embed messages in your server
Rename ;rename [NAME] This command changes the nickname of Sierra in your server
welcome message ;setwelcome Used to setup welcome message for your server
Leaving message ;setbye Command is used to setup and use leaving messages on your server
deleted message logging ;setdeletelog Used to setup deleted message logging in your server
Ban logging ;setbanlog Used to setup ban logging for your server
Announce command ;setannounce Used to setup the announcement command in your server
Clear command ;setclear Used to setup the clear command for your server
Admin tools ;setadmintools Used to enable kick, ban and unban commands in the server
Auto role ;setautorole Used to setup autorole for your server


Sierra is always being updated and improved so more commands will be added as updates for Sierra are released
