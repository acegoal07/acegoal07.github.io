General commands

Name Description Usage
Help ;help This command can be used in a chat to show all available commands while in discord
Server info ;serverinfo This server will show all the info about the server you will need to know
Server invite ;serverinvite This command will send an invite to the current text channel which can be shared outside of the server
Bot info ;botinvite This command will show you all the info you will need to know about Sierra
Roles ;roles This command will display all roles on the server and which ones you have
User info ;userinfo or ;userinfo [MENTION] Will display all needed information about a user
Report ;report [ERROR OR ISSUE] This command logs the issue so that i can find and fix the problem

Admin commands

Name Description Usage
Announcement ;announce [MESSAGE] This command takes the message and posts it in the announcement channel
Clear ;clear [1 to 100] Deletes the amount of messages that is defined
Rename ;rename [NAME] Changes the nickname of the discord bot on the server
Reload ;reload [COMMAND] Will reload the command that is specified
Kick ;Kick [MENTION] [REASON] Will kick the mentioned user
Log message ;logmessage [MESSAGE] Will display a message on the bot creators screen

Sierra is always being updated and improved so more commands will be added as these updates are added.
